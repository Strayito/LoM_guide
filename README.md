# Legend of Mushroom Guide

Welcome to the Legend of Mushroom guide! This website is dedicated to providing players with all the information they need to navigate and enjoy the game.

## Project Structure

The project consists of the following files and directories:

- **index.html**: The main entry point of the website, containing the layout and structure of the homepage.
- **guide.html**: A detailed guide for the game "Legend of Mushroom," including instructions and tips for players.
- **assets/**: Contains all the assets for the website.
  - **css/**: Contains the styles for the website.
    - **style.css**: Defines the visual appearance, including layout, colors, and fonts.
  - **js/**: Contains the JavaScript code for the website.
    - **main.js**: Includes functionality for interactive elements and dynamic content updates.
- **README.md**: This documentation file, providing setup instructions and usage guidelines.
- **CNAME**: Used for custom domain configuration on GitHub Pages.

## Setup Instructions

1. Clone the repository to your local machine:
   ```
   git clone https://github.com/yourusername/legend-of-mushroom-guide.git
   ```
   
2. Navigate to the project directory:
   ```
   cd legend-of-mushroom-guide
   ```

3. Open `index.html` in your web browser to view the homepage.

4. For the game guide, navigate to `guide.html`.

## Usage Guidelines

- Use the navigation links on the homepage to access different sections of the guide.
- The guide provides tips, strategies, and detailed information about gameplay mechanics.
- Feel free to contribute by submitting issues or pull requests if you have suggestions or improvements.
